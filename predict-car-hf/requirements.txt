ARAÇ FİYAT TAHMİN UYGULAMASI (README)
Bu proje, makine öğrenmesi kullanılarak araçların çeşitli özelliklerine göre fiyat tahmini yapan bir web uygulamasıdır. Hugging Face Spaces üzerinde çalıştırılmak üzere yapılandırılmıştır.

GEREKSİNİMLER
Python Sürümü: 3.13
Bağımlılıklar (requirements.txt):

scikit-learn==1.6.1

pandas

xlrd>1.0.0

gradio

joblib

DOSYA YAPISI
app.py               : Gradio arayüzünü başlatan ana uygulama dosyası.

car_price_model.pkl  : Eğitilmiş makine öğrenmesi modeli.

cars.xls             : Arayüzdeki seçenekleri (marka, model vb.) dinamik dolduran veri seti.

requirements.txt     : Proje bağımlılıklarını içeren dosya.

KURULUM VE ÇALIŞTIRMA
Gerekli kütüphaneleri yükleyin:
pip install -r requirements.txt

Uygulamayı başlatın:
python app.py

Hugging Face Spaces ortamında dağıtım yaparken, projenin ana dizininde bulunan requirements.txt dosyası otomatik olarak algılanıp derlenecektir. app.py dosyasının en üstüne YAML yapılandırmasının (frontmatter) eklenmiş olması gerekmektedir.
